# ACM-Template

> [English](./README.md)

## 安装

#### 前提:安装好 xelatex

> ### 克隆仓库
>
> ```shell
>  git clone https://github.com/HKing147/ACM-Template.git
> ```
>
> ### 安装字体
>
> #### 主要字体:Latin Modern Mono 和华文楷体,代码字体为 Consolas,代码注释字体为 Latin Modern Mono
>
> ```shell
> cd ACM-template/.fonts/
> cp ./* /usr/share/fonts/
> mkfontscale
> mkfontdir
> fc-cache
> ```
>
> ### 生成 pdf 模板文件
>
> ```shell
> cd ..
> make
> ```
